# Freshmen-Year-Projects
This repository contains code for the projects I built for my freshmen year courses: CS-109 and CS-111.
The scientific calculator performs the following functions:
 - +,-,*,/
 - HCF
 - sin, cos, tan, sin-inverse, cos-inverse, tan-inverse,
 - exponential
 - unit conversions: Celsius to Fahrenheit, kg to lb
 - logarithms
